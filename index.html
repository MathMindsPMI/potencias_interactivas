¿QUÉ ES UNA POTENCIA?

Una expresión matemática que representa la ✨ multiplicación de un número por sí mismo ✨ un cierto número de veces

BASE

Es el número que se multiplica por sí mismo

23

Esta es la BASE

EXPONENTE

Indica cuántas veces se multiplica la base por sí misma

23

Este es el EXPONENTE

TIPOS DE POTENCIAS

POSITIVA

El exponente es un número positivo

32 = 9

3 × 3 = 9

NEGATIVA

El exponente es un número negativo

2-2 = 1/4

1/(2²) = 0.25

CERO

El exponente es cero

50 = 1

Siempre es 1
